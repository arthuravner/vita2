Vita 2.0 teste

Em "Turmas" é possível fazer todo o gerenciamento das turmas e dos links das pesquisas de satisfação

É permitido buscar as turmas por "Nome", "Ano" e "Período"

Para buscar basta preencher os campo desejados e "Aplicar filtro"

Para criar uma nova turma, "Nova turma"

Para cada turma é possível gerenciar os links de pesquisa de satisfação e, editar e excluir a turma

Para gerenciar os links de pesquisa de satisfação basta clicar no primeiro botão de ação

Para editar uma turma, clicar no segundo botão de ação

Para excluir uma turma, clicar no terceiro botão de ação

Após criar uma turma é possível gerar um link de pesquisa de satisfação clicando em "Novo link"

É possível buscar os links já criados filtrando por descrição e "Aplicar filtro"

A "Data de validade" é para indicar até quando o link ficará disponível

Ao "Exigir identificação" o aluno deverá se identificar ao responder a pesquisa

Ao gerar o link essa tela será fechada e o link será exibido como primeiro item do grid

A coluna "Validade" indica até quando o link estará disponível

O botão "Copiar link" copia o link da pesquisa de satisfação para ser repassado aos alunos

A coluna "Resposta" indica a quantidade de alunos que já responderam a pesquisa

O primeiro botão de ação edita o link

O segundo botão de ação direciona o usuário para a página "Satisfação", onde é possível ver os resultados da pesquisa

É necessário selecionar um link de pesquisa de satisfação para a aplicar o filtro.

Após aplicar o filtro os resultados da pesquisa serão exibidos para aquele link

A classificação indica a nota dada pelo aluno para a aula

A satisfação indica o quão satisfeito o aluno está co ma aula

O índice de satisfação é obtido a partir da análise de sentimento do comentário feito pelo aluno sobre a aula

O botão de detalhe permite visualizar o texto do comentário

Na página "Dashboard" é possível visualizar os resultados das pesquisas de uma turma em forma de gráficos

No filtro basta selecionar a turma e "Aplicar filtro"
